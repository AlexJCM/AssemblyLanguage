## **EJERCICIOS DE PROGRAMACION EN LENGUAJE ENSAMBLADOR RESUELTOS UTILIZANDO NASM**

Este repositorio contiene 13 ejercicios de programación resueltos descritos a continuación utilizando Nasm.

**Ejercicio 01.-** Operaciones básicas con dos números <br/>
**Ejercicio 02.-** Calcular el área de un cuadrado <br/>
**Ejercicio 03.-** Calcular el promedio de dos examenes <br/>
**Ejercicio 04.-** Verificar si un número es positivo o negativo <br/>
**Ejercicio 05.-** Calcular el mayor de dos números <br/>
**Ejercicio 06.-** Calcular el mayor de 3 números <br/>
**Ejercicio 07.-** Calcular mayor de 2 números y sumar o restar según sea el caso <br/>
**Ejercicio 08.-** Dividir dos números <br/>
**Ejercicio 09.-** Determinar si un triángulo es equilátero <br/>
**Ejercicio 10.-** Si de A y B uno de ellos es positivo multiplicarlos <br/>
**Ejercicio 11.-** Presentar el día de la semana ingresando su respectivo número <br/>
**Ejercicio 12.-** Calculadora con menú de operaciones <br/>
**Ejercicio 13.-** Leer 3 números y comparar si A+B = C <br/>
