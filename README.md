## **EJERCICIOS DE PROGRAMACION EN LENGUAJE ENSAMBLADOR RESUELTOS UTILIZANDO NASM**

Este repositorio contiene 13 ejercicios de programación resueltos descritos a continuación utilizando Nasm.

**Ejercicio 01.-** Operaciones básicas con dos números
**Ejercicio 02.-** Calcular el área de un cuadrado
**Ejercicio 03.-** Calcular el promedio de dos examenes
**Ejercicio 04.-** Verificar si un número es positivo o negativo
**Ejercicio 05.-** Calcular el mayor de dos números
**Ejercicio 06.-** Calcular el mayor de 3 números
**Ejercicio 07.-** Calcular mayor de 2 números y sumar o restar según sea el caso
**Ejercicio 08.-** Dividir dos números
**Ejercicio 09.-** Determinar si un triángulo es equilátero
**Ejercicio 10.-** Si de A y B uno de ellos es positivo multiplicarlos
**Ejercicio 11.-** Presentar el día de la semana ingresando su respectivo número
**Ejercicio 12.-** Calculadora con menú de operaciones
**Ejercicio 13.-** Leer 3 números y comparar si A+B = C
